Instructions

This interface allows you to construct, view, and publish a custom presentation of a protein structure. The presentation should be divided into multiple frames. In each frame, you should highligh a structural feature of the protein and discuss its biological implications.

Edit Frames

Use this menu to add, edit, and delete frames. Controls:

  • Click the Add Frame button to add a new frame.
  • Click the Save orientation button to save changes you have made to a frame. If you do not click this button before going to a different frame (clicking Add Frame will do this) then any changes you make will not be saved to the presentation.
  • The Frames Selector lets you click on any frame in order to view and edit it.
  • Click the Delete Frame button to permanently delete the frame that you are currently viewing from the presentation.
  • The Frame Message allows you to edit the message associated with each frame. The message should describe the structural feature that you are highlighting in each frame and discuss its biological implications.

Edit Display

  • The Display Residues function allows you to make the atomic structure of specific residues visible in a frame.
  • The chain selector allows you to choose a specific chain to highligh residues in. Currently this application does not support displaying residues from different chains in the same frame.
  • The residues selector allows you to choose which residues to display. Simply click on the residue to make it visible in the frame.

Preview Presentation

Click this tab to see a what other site users will see when they view your presentation.

Publish Presentation

Click this tab to add a title, description, and your name to your presentation and then to upload it to the website for other users to view.

Clicking Submit will upload the presentation and then redirect you to the public view of your presentation. Currently this application does not support editing presentations after submitting them.
