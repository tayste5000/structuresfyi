About

What's the point of structures.fyi?

This website was made with two uses in mind:

  • Giving biochemists an interactive, immersive, and easy to use platform for presenting structural information
  • Giving laypeople a peak into the fun and interesting world of structural biology

Who made structures.fyi?

Structures.fyi is a personal project of Taylor Stevens, who is a PhD student in Biochemistry and Molecular Biophysics at Caltech.

Are there bigger plans down the road?

The current version of the website was made to test the water for interest in a more complete application. I have a number of ideas for features that could make this website more useful, and I am hoping that I receive a lot of input on the feedback page to get a better idea of what people might want out of structures.fyi. Some ideas that I have:

  • More flexibility control and edit the input structure (i.e. uploading custom .pdb files, combining multiple files, translating individual protein chains).
  • Add user accounts and give users the option to create non-public structure presentations.
  • Add voting features to allow users to vote on a 'Structure of the Month' or something like that.
  • Provide presentations in an embedable format so that labs, journals, and anyone else can add presentations to their own webpages.
  • Add features that facilitate sharing presentations through social media.
